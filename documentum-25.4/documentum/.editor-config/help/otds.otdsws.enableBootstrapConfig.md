### Configure OTDS automatically

<!-- Variable Description -->
Configure OTDS automatically using documentum\charts\otds\charts\otdsws\config\config.yml file. 

#### Variable Location
```yaml
# documentum/config/configuration.yml
otds:
  otdsws:
    enableBootstrapConfig: true
```

<br>

#### Additional Information
* **Note**: The config.yml file contains initial configuration information when the database is populated.
