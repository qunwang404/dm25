### Client Configuration Web App Title

<!-- Variable Description -->
Indicates the custom applcation title that gets displayed on the Browser tab.

#### Variable Location
```yaml
# documentum/config/configuration.yml
d2config:
  env:
    appTitle: D2-Config
```
<br>

#### Additional Information
* **Default Value**: `D2-Config`
* **Note**: Configure the application title in the client configuration project.
