### Admin Console New Relic App Name

<!-- Variable Description -->
Name of the Admin Console pod displayed in the New Relic dashboard. This variable is used to show the Admin Console metrics in the New Relic dashboard.

#### Variable Location
```yaml
# documentum/config/configuration.yml
adminconsole:
  newrelic:
    app_name: DOCUMENTUM_ADMINCONSOLE-PROD-OT2_BP2-ECD
```
