### OTDS Endpoint for SOAP Config

<!-- Variable Description -->
OTDS endpoint for SOAP config. This variable is related to OpenText Content Management.

#### Variable Location
```yaml
# documentum/config/configuration.yml
dctm-cssap:
  containers:
    cssap:
	  soapConfigOtdsEndpoint: http:\\/\\/<ip-host>:<port>\\/otdsws\\/services\\/Authentication
```
