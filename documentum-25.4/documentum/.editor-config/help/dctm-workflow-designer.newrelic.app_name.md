### Workflow Designer New Relic App Name

<!-- Variable Description -->
Workflow Designer pod name in the New Relic dashboard. This name is used to display the Workflow Designer metrics in the New Relic dashboard.

#### Variable Location
```yaml
# documentum/config/configuration.yml
dctm-workflow-designer:
  newrelic:
    app_name: DCTM_WFD-PROD-OT2_CFCR_LI3-EIM
```
