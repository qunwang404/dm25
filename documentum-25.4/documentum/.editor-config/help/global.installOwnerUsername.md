### Installation Owner User Name

<!-- Variable Description -->
User name of the installation owner.

#### Variable Location
```yaml
# documentum/values.yaml
global:
  installOwnerUsername: dmadmin
```
<br>

#### Additional Information
* **Default Value**: `dmadmin`
* **Constraints**: Must be a valid user name.
* **Note**: The user `dmadmin` is defined as the installation owner.
