### Connector for Core Share URL

<!-- Variable Description -->
URL for Documentum Connector for Core Share.

#### Variable Location
```yaml
# documentum/config/configuration.yml
d2smartview:
  D2FS:
    dccUrl:
```
<br>

#### Additional Information
* **Default Value**: Null
* **Note**: This URL is used to interact with the D2FS services, providing functionalities such as document management, workflow operations, and other content services 
            within the Smart View application. You must restart the application server if you have made changes to this variable.
