### S3 Store Base URL

<!-- Variable Description -->
URL that Documemtum CM Server uses to communicate with the Amazon S3 store.

#### Variable Location
```yaml
## documentum/config/passwords.yaml OR documentum/config/passwords_vault.yaml or documentum/config/passwords_k8api.yaml
cs-secrets:
  s3Store:
    s3StoreBaseUrl:
```

<br>

#### Additional Information
* **Note**: For more information about the Amazon S3 store, see OpenText Documentum Content Management - Server Administration and Configuration Guide.
